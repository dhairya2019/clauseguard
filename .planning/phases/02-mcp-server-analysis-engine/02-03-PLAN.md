Create `README.md` with the following sections:

**Header:**
- Title: ClauseGuard MCP Server
- One-line description: "MCP server for legal contract analysis with India-specific awareness, powered by Claude"
- Badge ideas: npm version, license MIT

**Features (bullet list):**
- Clause-by-clause risk assessment (HIGH/MEDIUM/LOW)
- India-specific legal flags (S.27 non-compete, S.28, FEMA, jurisdiction)
- Missing clause detection per document type
- Safer alternative wording suggestions
- Structured JSON output
- Supports freelance agreements, NDAs, ToS, employment, SaaS contracts

**Prerequisites:**
- Node.js 18+
- Anthropic API key (link to console.anthropic.com)

**Quick Start:**
```bash
# Set your API key
export ANTHROPIC_API_KEY=sk-ant-...

# Run via npx (no install needed)
npx -y clauseguard-mcp
```

**Claude Desktop Configuration:**
```json
{
  "mcpServers": {
    "clauseguard": {
      "command": "npx",
      "args": ["-y", "clauseguard-mcp"],
      "env": {
        "ANTHROPIC_API_KEY": "sk-ant-..."
      }
    }
  }
}
```
Note: Config file location is `~/Library/Application Support/Claude/claude_desktop_config.json` on macOS.

**Claude Code Configuration:**
```bash
claude mcp add clauseguard -- npx -y clauseguard-mcp
```
Note: Set ANTHROPIC_API_KEY in your shell profile.

**Tool: analyze_contract**
- Inputs: contract_text (required, min 50 chars), analysis_type (full/risk/summary, default: full), party_perspective (optional)
- Output: JSON with documentType, riskScore, riskBreakdown, clauses array, missingClauses, disclaimer

**Configuration (env vars):**
- `ANTHROPIC_API_KEY` (required) — Your Anthropic API key
- `CLAUSEGUARD_MODEL` (optional) — Claude model to use (default: claude-sonnet-4-5-20250514)

**Development:**
```bash
git clone ... && cd mcp-server
npm install
npm run dev          # Run with tsx
npm run build        # Compile TypeScript
npm run inspect      # Open MCP Inspector
```

**License:** MIT
